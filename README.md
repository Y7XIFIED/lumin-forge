# Lumin Forge

> A cinematic React experience forged for immersive digital identity.

![Build](https://img.shields.io/badge/build-passing-22c55e)
![License: MIT](https://img.shields.io/badge/license-MIT-111111)
[![Tag](https://img.shields.io/github/v/tag/Y7XIFIED/lumin-forge)](https://github.com/Y7XIFIED/lumin-forge/tags)

## Features

- Cinematic motion-driven single-page experience
- Modular React + TypeScript architecture
- Tailwind-powered styling with custom typography
- GSAP + Framer Motion animation system
- Vite-powered fast local development and builds

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
