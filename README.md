## Overview
Lumin Forge is a motion-led brand experience built for immersive digital storytelling.  
It combines a cinematic visual system with modular components, smooth interactions, and fast rendering.
The project is designed to feel premium while staying straightforward to develop and maintain.

Live site: [https://lumin-forge.vercel.app](https://lumin-forge.vercel.app)

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion + GSAP

## Features
- Cinematic one-page flow built for visual narrative and strong brand presence
- Smooth animated transitions and reveal effects across key sections
- Modular component architecture for fast iteration and easy customization
- Responsive behavior for desktop and mobile viewports
- Lightweight developer workflow with Vite-based local and production builds
- Ready for static deployment on platforms like Vercel

## Installation
1. Clone the repository.

```bash
git clone https://github.com/Y7XIFIED/lumin-forge.git
```

2. Move into the project directory.

```bash
cd lumin-forge
```

3. Install dependencies.

```bash
npm install
```

4. Run the local development server.

```bash
npm run dev
```
